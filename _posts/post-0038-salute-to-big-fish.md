先用 VLC 播放器截屏，再用下面的脚本批处理图片：

```bash
#!/bin/bash

X="75"
Y="79"
dX="+0"
dY="+0"

for i in *.png;
do
    image=`basename ${i} .png`
    echo Input: ${i}
    convert ${i} -gravity Center -crop ${X}x${Y}%${dX}${dY} ${i} && echo "Croping done"
    convert ${i} -resize %50 ${i} && echo "Resizing done"
done

echo ===============
echo all done
```
