1447590832: 赋值表达式
----------------------

Assignment Expression

:   An assignment expression has the value of the left operand after the assignment.

    出自 C99 standard, section 6.5.16

有赋值表达式的情况下

:   ```python
    if result=func1(...):
        print func2(result)
    ```

无赋值表达式的情况下

:   ```python
    result=func1(...)
    if result:
        print func2(result)
    ```

然后你就知道什么叫赋值表达式了。
