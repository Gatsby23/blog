本博基本上分离在两条流水线：

- Markdown + Vim + Command line，这边用起来比较简单；
- org-mode + Emacs (batteries included)，这边功能强大，可以加密一些私人的东西。
  而且 org-mode 支持文学化编程（或者说程序化文档）。
